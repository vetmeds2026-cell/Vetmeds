## 🧩 The Problem

India has **65+ million pet-owning households** — a number growing 20% year over year. Yet:

- 🏥 Most cities have fewer than 5 qualified veterinary clinics per 100,000 people
- 🚨 There is **no standardized emergency alert system** between pet owners and vets
- 📋 Pet medical records are scattered across paper files, WhatsApp messages, and memory
- 🤷 Pet owners rely on unverified Google searches for health guidance — often dangerously so
- ⏳ Booking appointments requires phone calls, waiting, and back-and-forth

**The result?** Delayed care, misinformed decisions, and preventable pet deaths.

---

## 💡 Our Solution

**VetMeds** is a full-stack veterinary health management platform that makes professional pet care accessible to every pet owner — digitally, instantly, and affordably.

We bring together:
- 📲 **Smart appointment booking** with real vets
- 🚨 **One-tap emergency SOS** with live location and photo
- 🤖 **AI-powered health guidance** available 24/7
- 📋 **Digital pet health records** in one place
- 💊 **Digital prescriptions** from verified doctors

## 🏗️ How It's Built

VetMeds is production-ready, built on a modern, scalable stack:

```
Frontend          →  Next.js 15 (App Router) + React 19 + Tailwind CSS v4
UI Components     →  shadcn/ui + Framer Motion
Maps              →  MapLibre GL
PDF Generation    →  jsPDF (client-side, no server cost)
Database          →  Neon Serverless PostgreSQL + Drizzle ORM
Authentication    →  Clerk (OAuth, protected routes, user management)
AI Chatbot        →  Google Gemini AI
Image Storage     →  Cloudinary
Email             →  Nodemailer (SMTP) + Resend + EmailJS
Deployment        →  Vercel (zero-downtime, global CDN)
```

> **Why this stack?** Every technology choice was made for scalability, speed, and cost-efficiency at the MVP stage — serverless database, edge-deployed frontend, and pay-as-you-go AI calls mean near-zero infrastructure cost until we hit real scale.

## 🌊 Emergency SOS — The Killer Feature

Most veterinary platforms stop at booking. VetMeds goes further with a **real-time emergency response system**:

```
Pet owner taps SOS
        ↓
Describes the emergency + uploads a photo
        ↓
Pins exact GPS location on interactive map
        ↓
Alert saved to database instantly
        ↓
Doctor receives email with photo + Google Maps link + full description
        ↓
Doctor opens SOS dashboard → responds → marks resolved
        ↓
Owner sees resolution status in their SOS history
```

This solves the most critical gap in pet healthcare — **the gap between "something is wrong" and "a vet knows about it."**
